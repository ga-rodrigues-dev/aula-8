## Skills
| Skill | Técnica | Quando usar |
|---|---|---|
| `teste-unitario-funcional` | Partição de Equivalência + Análise de Valor-Limite | Testes baseados em especificação |
| `teste-unitario-cobertura` | Cobertura estrutural (JaCoCo) | Completar/evoluir suíte existente |
| `teste-sistema-playwright` | E2E com Playwright | Fluxos de negócio pela UI |

## Como usar
Carregue o SKILL.md desejado como instrução/skill na ferramenta de IA e
indique a classe/fluxo alvo. Cada Skill define regras, fluxo de trabalho
e formato de saída com explicabilidade (rastreabilidade teste → critério).
